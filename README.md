# 📖 Overview

The **Smart Library Management System** is a web application that simplifies library operations by providing an efficient way to manage books, students, journals, and book transactions.

It helps librarians automate routine tasks such as issuing books, returning books, searching the catalog, and managing user requests.

# 🚀 Installation

### 1️⃣ Install XAMPP

Download and install XAMPP.

### 2️⃣ Copy Project

Copy the project folder into:

```text
xampp/htdocs/
```

### 3️⃣ Start Services

Start:

- Apache
- MySQL

### 4️⃣ Import Database

Open **phpMyAdmin**

Create a database and import:

```text
database/librarydetails.sql
```

### 5️⃣ Run Project

Open your browser:

```text
http://localhost/Smart-Library-Management/
```
